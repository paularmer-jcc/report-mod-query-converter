Report Mod Query Converter

This tool converts SQL from BI Query User to a format for Power BI Report Builder.

To use, paste the SQL from BI Query User in the Input text box, then click Convert. You will be prompted for variable names found in the input SQL. Clicking 'OK' without providing any input will take the suggestion provided in (parenthesis). After processing all variables, the result will appear in the Result text box.
